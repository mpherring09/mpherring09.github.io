## **What is ELO?**

ELO is a rating system used primarily in chess, which rewards players for upsetting highly ranked opponents, and penalizes them when the lose to lower ranked opponents. This system was adapted by collegefootballdata.com to rate football teams, additionally accounting for margin of victory and roster turnover.

## **ELO Graph Project:**

I used this dataset to produce an interactive graph which displays 100 years of CFB history. Some of the challenges of this project included:
- Manipulating and cleaning the datasets using the Pandas library
- Visualizing thousands of games for over 100 teams
- Transforming the graph view and labelling based on user input
- Optimizing the data processing to allow for real-time animation in a web browser
